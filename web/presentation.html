Weighted Interval Scheduling

Given a set of jobs J

Job i âˆˆ J has a start time si, a finish time fi, and a weight wi

Find the maximum sum of weights for a subset O of non-overlapping jobs

case class Job(s: Int, f: Int, w: Int)

def optimumSchedule: List[Job] => Int = {
  case Nil => 0
  case i :: rest =>
    val nonOverlaps = rest.filterNot(i.overlaps)
    max(optimumSchedule(nonOverlaps) + i.w,
        optimumSchedule(rest))
}
